### Description:
The first study project on Python Developer course (by Hexlet.io)
It contains a few elementary games 3 rounds each. Make no mistake to win. Description of an each one is given below

### Install:
To install and play games use commands from makefile commands.
```jsx
make install
make publish
make package-install
make build

```

### Calculator:
player should evaluate ordinary arithmetical operation.Type the following to start:
```jsx
brain-calc
```
