### Description:
The first study project on Python Developer course (by Hexlet.io)
It contains a few elementary games 3 rounds each. Make no mistake to win.

brain-calc: player should evaluate ordinary arithmetical operation
brain-even: player should guess wether the nuber is even or not
brain-gcd: player shoud find the greatest common divider of two suggested numbers
brain-prime: player should guess wether the nuber is prime or not
brain-progression: player should find the skipped number in arithmetical progression
